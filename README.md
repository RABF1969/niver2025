## 🚀 Tecnologias

- **React 18** com **TypeScript**
- **Vite** para build e desenvolvimento
- **TailwindCSS** para estilização
- **React Router** para navegação
- **Design responsivo** (Mobile-first)
- **Dark Mode** incluído

```bash
# Clonar o repositório

# Entrar no diretório
